# Data Analysis with PostgreSQL
PostgreSQL offers numerous subprograms and keywords to make it quite easy of developer experience / database use experience / DBA experience / data analyst experience. ;)
Some of them are quite popular nowadays, including:
- COUNT()
- ROUND()
- PERCENTILE_CONT()
- MODE()
- RANK()
- CAST()
- MAX()
- PERCENTILE_CONT() WITHIN GROUP
- MAX
- MIN
- AVG
- STDDEV_SAMP
- ROUND(... OVER (ORDER BY ... ROWS BETWEEN m PRECEDING AND n FOLLOWING))
- MAX(...) OVER (ORDER BY --- ROWS BETWEEN m PRECEDING AND CURRENT ROW)
- SUM(... ) OVER (PARTITION BY extract(... FROM ...), extract(... FROM ...) ORDER BY ... ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW
